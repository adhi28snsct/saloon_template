# AGENT.md

## Project Context

This is a Salon Management Dashboard built using Next.js (App Router) and shadcn/ui.

Primary goal:
👉 Use as a demo to acquire clients and convert into paid customization projects.

---

## 🎯 Purpose

- Demo-first project
- UI-focused
- Fast to modify for different industries

---

## 🧩 Core Modules

### 1. Dashboard
- Summary of appointments
- Follow-up reminders
- Daily overview

### 2. Appointments
- Booking list
- Status tracking (Booked / Completed)

### 3. Customers
- Customer details
- Visit history

### 4. Follow-ups
- Reminder tracking
- Retention system

---

## ⚙️ Tech Guidelines

- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Components: shadcn/ui
- Icons: lucide-react

---

## 🧠 Development Principles

- Keep components reusable
- Use server components where possible
- Avoid overcomplication
- Prioritize speed over perfection
